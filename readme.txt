Plugin, který umožňuje snadno vložit shortcode s URL na [ČSFD.cz](https://www.csfd.cz/), který pak bude zobrazen jako snippet s informacemi o filmu na vašich stránkách. Od verze __0.3.0__ pak přidává i vlastní typ příspěvků, kdy je možno vytvořit nový příspěvek z URL odkazující na [ČSFD.cz](https://www.csfd.cz/).
